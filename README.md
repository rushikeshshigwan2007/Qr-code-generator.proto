# QR Code Generator

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/rushikeshshigwan2007/QR-Code-Generator/blob/main/QR_Code_Generator.ipynb)

A simple QR Code Generator built using Python in Google Colab.

## Features

- Generate a QR code from any URL.
- Save the generated QR code as a PNG image.
- Beginner-friendly and easy to use.
- Runs directly in Google Colab.

## Requirements

- Python 3
- Google Colab
- qrcode library

## Installation

```python
!pip install qrcode
```

## How to Run

1. Open the notebook in Google Colab using the button above.
2. Run the installation cell.
3. Run the code cell.
4. Enter the URL when prompted.
5. The QR code will be generated and saved as `qrcode.png`.

## Output

The generated QR code is saved in:

```
/content/qrcode.png
```
